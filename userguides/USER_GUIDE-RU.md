# 📕 ST Memory Books — Ваш ИИ-ассистент памяти

**Превратите бесконечные переписки в организованные, доступные для поиска воспоминания!**

Нужно, чтобы бот помнил детали, но контекст чата слишком велик? Хотите автоматически отслеживать важные сюжетные повороты, не делая заметки вручную? **ST Memory Books** делает именно это — расширение наблюдает за вашими чатами и создает умные сводки (саммари), чтобы вы никогда больше не теряли нить истории.

---

## 🚀 Быстрый старт (Первое воспоминание за 5 минут)

**Впервые здесь?** Настроим автоматическое запоминание всего за пару кликов:

### Шаг 1: Найдите расширение

* Найдите иконку волшебной палочки (🪄) рядом с полем ввода чата.
* Нажмите на нее, затем выберите **"Memory Books"**.
* Откроется панель управления ST Memory Books.

### Шаг 2: Включите автомагию

* В панели управления найдите пункт **"Auto-Summary"**.
* Переключите его в положение **ON** (Вкл).
* Установите создание воспоминаний каждые **20-30 сообщений** (хорошее значение для начала).
* Готово! 🎉

### Шаг 3: Общайтесь как обычно

* Просто продолжайте переписку.
* Спустя 20-30 новых сообщений ST Memory Books автоматически:
* Подберет лучшие границы сцены.
* Попросит ваш ИИ написать саммари.
* Сохранит его в коллекцию воспоминаний (лорбук).
* Покажет уведомление о завершении.



**Поздравляем!** Теперь у вас работает автоматическое управление памятью. Больше не нужно вспоминать, что случилось несколько глав назад!

## 💡 Основные возможности

Думайте о ST Memory Books как о вашем **личном ИИ-библиотекаре**:

### 🤖 **Автоматические сводки (Auto-Summary)**

*"Я не хочу думать об этом, пусть просто работает"*

* Наблюдает за чатом в фоновом режиме.
* Автоматически создает записи каждые X сообщений.
* Идеально для длинных ролевых игр (RP) и бесконечных историй.

### ✋ **Ручное создание (Manual Creation)**

*"Я хочу сам решать, что сохранить"*

* Отмечайте важные сцены с помощью кнопок-стрелок (► ◄) прямо в чате.
* Создавайте воспоминания по требованию для особых моментов.
* Отлично подходит для фиксации ключевых поворотов сюжета.

### 📊 **Фоновые трекеры и Шаблоны (Side Prompts)**

*"Я хочу отслеживать отношения, квесты или статистику"*

* Используйте шаблоны для отслеживания динамики отношений, настроения или инвентаря.
* Автоматически обновляет "табло счета", статус квестов или психологический портрет.
* Примеры: "Кто кому нравится?", "Текущая цель миссии", "Настроение персонажа".

### 📚 **Коллекции воспоминаний**

*Где хранятся все записи*

* Автоматически организуются и доступны для поиска.
* Полная интеграция с системой Лорбуков (World Info) в SillyTavern.
* Ваш ИИ может ссылаться на прошлые воспоминания в новых диалогах.

## 🎯 Выберите свой стиль

### 🔄 "Поставил и забыл" (Рекомендуется новичкам)

1. Включите **"Auto-Summary"** в настройках.
2. Выберите частоту (каждые 20-50 сообщений).
3. Общайтесь — воспоминания создаются сами!

### ✋ "Ручной контроль" (Для избирательных)

1. Ищите кнопки-стрелки (► ◄) на сообщениях в чате.
2. Нажмите **►** на первом сообщении важной сцены.
3. Нажмите **◄** на последнем сообщении этой сцены.
4. Откройте Memory Books (🪄) и нажмите **"Create Memory"**.

### ⚡ "Power User" (Слэш-команды)

Для любителей клавиатуры и макросов:

* `/scenememory 10-25` — Создать память из сообщений с 10 по 25.
* `/creatememory` — Создать память из текущей отмеченной сцены.
* `/nextmemory` — Суммировать всё с момента последнего воспоминания.
* `/sideprompt "Relationship Tracker" {{macro}}="value"` — Запустить конкретный трекер.
* После выбора Side Prompt автодополнение slash-команд предложит оставшиеся обязательные runtime-макросы.

## 🌈 Сюжетные Арки (Arc Summaries)

Сюжетные Арки помогают управлять очень длинными историями, объединяя старые воспоминания.

* **Зачем это нужно?** Со временем накапливается много мелких записей. Арки позволяют объединить несколько старых воспоминаний в одно общее саммари, экономя токены и упрощая контекст.
* **Как это работает?** Старые записи объединяются в одну новую, а оригиналы скрываются (но не удаляются).

**Как создать Арку:**

1. Нажмите **🌈 Consolidate Memories into Arcs** внизу главного окна расширения.
2. Выберите тип арки (Multi-Arc, Single Arc или Tiny).
3. Выберите воспоминания для объединения и нажмите **Run**.

## 🎨 Трекеры и Шаблоны (Advanced)

**Side Prompts (Фоновые трекеры)** создают отдельные записи Side Prompt в lorebook и работают параллельно с созданием воспоминаний. Они обновляют специфические заметки, не меняя основной промпт персонажа.
Стандартные макросы ST, такие как `{{user}}` и `{{char}}`, разворачиваются в `Prompt` и `Response Format`. Нестандартные `{{...}}`-макросы становятся обязательным вводом для ручного запуска.

* **Библиотека шаблонов:** В настройках есть готовые шаблоны (Character Development, Relationship Dynamics и др.).
* **Как использовать:** Просто включите нужный шаблон в менеджере Side Prompts.
* **Ручной режим:** Если в шаблоне есть пользовательские макросы, его нужно запускать вручную через `/sideprompt`.
* **Свой трекер:** Вы можете создать свой. Например: *"Всегда записывай текущую погоду в сцене"*.
* **Проверка безопасности:** Если шаблон содержит пользовательские runtime-макросы, STMB удаляет `onInterval` и `onAfterMemory` при сохранении/импорте и показывает предупреждение.

## 🧠 Продвинутый контроль текста (Regex)

ST Memory Books интегрируется с официальным расширением **Regex** для SillyTavern.

Это позволяет:

* **Чистить ответы ИИ:** Автоматически удалять фразы вроде *"As a large language model..."* или *"OOC:"* из создаваемых саммари.
* **Форматировать промпты:** Изменять текст истории перед тем, как он будет отправлен на суммаризацию.

Для настройки используйте Placement (Размещение): `User Input` для исходящих данных и `AI Output` для входящих саммари.

## ⚙️ Важные настройки

* **Auto-Summary Frequency:**
* 20-30: Для детальных, медленных чатов.
* 40-60: Для экшена и быстрых диалогов.


* **Memory Previews (Предпросмотр):** Включите, если хотите редактировать саммари перед сохранением.
* **Memory Titles:** Настройте формат названий (например, `"Глава {{title}}"`).
* **Memory Collections:** Режим **Auto** использует лорбук, привязанный к чату. Режим **Auto-create** создаст новый лорбук, если его нет.

## 🔧 Решение проблем (Troubleshooting)

**"Я не вижу опцию Memory Books!"**
Убедитесь, что расширение установлено и включено. Ищите иконку 🪄. Попробуйте обновить страницу.

**"Кнопки стрелок (► ◄) не появляются!"**
Они появляются через 3-5 секунд после загрузки чата. Если их нет, обновите страницу.

**"Авто-саммари не работает!"**
Проверьте, включено ли оно. Достигнут ли интервал сообщений? (Счетчик сбрасывается после каждого создания памяти).

**"Ошибки отсутствия лорбука (Missing lorebooks)"**
В настройках включите "Auto-create lorebook if none exists" или вручную привяжите лорбук к чату.

**"Иногда процесс падает без причины!"**
Убедитесь, что в пресете SillyTavern параметр **Max Response Length** достаточно велик (рекомендуется минимум 2000 токенов, лучше 4000).
